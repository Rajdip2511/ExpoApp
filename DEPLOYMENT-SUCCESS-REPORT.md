## 📱 **HOW TO RUN THE APPLICATION**

### Backend Server:
```bash
cd backend
npm install
npx prisma db push
npx prisma db seed
npm start
```
**Server will be available at:** http://localhost:4000

### Mobile App:
```bash
cd mobile-app
npm install
npm start
```
**Scan QR code with Expo Go app**
